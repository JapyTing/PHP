Formularios

Introduccion

Es un control creado con el tag FORM, el cual permite obtener informaciòn de entrada desde el usuario


Atributos importantes
  • ACTION: Establece la URL destino de los datos
  • METHOD: Establece el verbo del protocolo HTTP con el cual se enviaran la informaciòn. Los mas usados son GET y POST
  • NAME: Asigna un nombre al formulario para ser recuperado del lado del servidor

Tipos de entrada

  • Correo (EMAIL): Permite capturar correos electronicos validados
